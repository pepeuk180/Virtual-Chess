# JUEGO DEL AJEDREZ

## DESCRIPCIÓN

El juego del ajedrez es desarrollado para un caso de estudio, la empresa tiene trabajadores con bajos indices de trabajo y altos indices de estres, por lo que esta aplicación se encargara de subasanar las dificultades de los empleados con una actividad lúdica, que es el Ajedrez.

## CARACTERÍSTICAS

- Juego del ajedrez diseñado en la consola
- Modo de juego local
- Seleción de 8 colores distintos para las piezas
- Requiere de Java 17 o versiones actuales para funcionar
- Las funcionalidades de la aplicación estan basadas bajo el [reglamento del ajedrez](https://www.casi.com.ar/sites/default/files/Reglamento%20Ajedrez.pdf)

## USO

- Las coordenadas de origen y destino deben ser como: ***A6, h4, C2***; respetando la estructura del primer caracter debe ser una letra del ***A-H*** y el segundo caracter del ***1-8***
- Para solicitar tablas digita: ***1/2***
- Para realizar un enroque debera seleccionar su rey como coordenada de origen y para la coordenada de destino debe seleccionar la casilla en donde terminara el rey despues del enroque
